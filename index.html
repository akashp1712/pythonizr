About Pythonizr

Pythonizr is here to kick-start the development of your new Python project. It generates necessary boilerplate code to get started with. Also, provided a few helpers to add in a project.

Based on other *izr

This project is heavily inspired by Initializr by Jonathan Verrecchia and Extensionizr by Alex Wolkov
