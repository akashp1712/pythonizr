About Pythonizr

Pythonizr is here to kick-start the development of your ML project. It generates the basic code using for sci-kit library in python.

Based on other *izr

This project is heavily inspired by Initializr by Jonathan Verrecchia and Extensionizr by Alex Wolkov

But Why?

Because Time flies. This should ease the life a bit(!) and provide a good platform for the beginners ML engineers.
